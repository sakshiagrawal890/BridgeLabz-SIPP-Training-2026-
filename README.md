# BridgeLabz SIPP Training 2026

This repository contains Core Java practice programs.

## Branch

The Core Java practice work is available in the `corejavapractice` branch.

## Folder Structure

```text
core-java-practice/
└── gcr-codebase/
    └── recursions/
        ├── ArraySubsets.java
        ├── Factorial.java
        ├── FibonacciNumber.java
        ├── PalindromeCheck.java
        ├── PowerCalculation.java

```

## Recursion Programs

1. Print numbers from N to 1
2. Calculate factorial
3. Sum of first N natural numbers
4. Reverse a string
5. Check palindrome
6. Find nth Fibonacci number
7. Calculate power of a number
8. Generate all subsets of an array
9. Solve Tower of Hanoi
10. Generate all permutations of a string

## How to Run

Go to the recursion folder:

```bash
cd core-java-practice/gcr-codebase/recursions
```

Compile all programs:

```bash
javac *.java
```

Run any program:

```bash
java Factorial
```

Example input:

```text
5
```

Example output:

```text
120
```
